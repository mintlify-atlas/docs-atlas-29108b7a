## Overview

The Video resource represents video files for anime themes. Each video contains metadata about resolution, source, encoding properties, and is linked to theme entries.

## List Videos

<ParamField path="GET" type="endpoint">
  /api/video
</ParamField>

Retrieve a paginated list of videos.

### Query Parameters

<ParamField query="page[size]" type="integer">
  Number of results per page (default: 15, max: 100)
</ParamField>

<ParamField query="page[number]" type="integer">
  Page number to retrieve
</ParamField>

<ParamField query="include" type="string">
  Comma-separated list of relationships to include. Available includes:
  - `animethemeentries` - Theme entries this video belongs to
  - `animethemeentries.animetheme` - Anime themes
  - `animethemeentries.animetheme.anime` - Parent anime
  - `animethemeentries.animetheme.song` - Associated songs
  - `audio` - Audio track for this video
  - `videoscript` - Subtitle/script file
  - `tracks` - Playlist tracks using this video
</ParamField>

<ParamField query="filter[basename]" type="string">
  Filter by video basename
</ParamField>

<ParamField query="filter[resolution]" type="integer">
  Filter by resolution (e.g., 720, 1080)
</ParamField>

<ParamField query="filter[source]" type="string">
  Filter by source. Values: `WEB`, `RAW`, `BD`, `DVD`, `VHS`, `LD`, `AT`
</ParamField>

<ParamField query="filter[nc]" type="boolean">
  Filter by creditless (NC) status. Values: `true`, `false`
</ParamField>

<ParamField query="filter[subbed]" type="boolean">
  Filter by subbed status. Values: `true`, `false`
</ParamField>

<ParamField query="filter[lyrics]" type="boolean">
  Filter by lyrics status. Values: `true`, `false`
</ParamField>

<ParamField query="filter[uncen]" type="boolean">
  Filter by uncensored status. Values: `true`, `false`
</ParamField>

<ParamField query="filter[overlap]" type="string">
  Filter by overlap type. Values: `None`, `Transition`, `Over`
</ParamField>

<ParamField query="fields[video]" type="string">
  Comma-separated list of fields to return. Available fields:
  - `video_id`
  - `basename`
  - `filename`
  - `path`
  - `size`
  - `mimetype`
  - `resolution`
  - `source`
  - `nc`
  - `subbed`
  - `lyrics`
  - `uncen`
  - `overlap`
  - `audio_id`
  - `link`
  - `tags`
  - `priority`
</ParamField>

<ParamField query="sort" type="string">
  Sort by field. Prefix with `-` for descending order.
  Available: `basename`, `resolution`, `source`, `created_at`, `updated_at`
</ParamField>

<ParamField query="q" type="string">
  Full-text search query across video basenames and anime
</ParamField>

### Response

<ResponseField name="videos" type="array">
  Array of video objects
  
  <ResponseField name="video_id" type="integer">

  <ResponseField name="basename" type="string">

  <ResponseField name="filename" type="string">

  </ResponseField>
  
  <ResponseField name="path" type="string">
    Full storage path
  </ResponseField>
  
  <ResponseField name="size" type="integer">
    File size in bytes
  </ResponseField>
  
  <ResponseField name="mimetype" type="string">
    MIME type (e.g., "video/webm")
  </ResponseField>
  
  <ResponseField name="resolution" type="integer" optional>
    Video resolution (e.g., 720, 1080)
  </ResponseField>
  
  <ResponseField name="source" type="string" optional>
    Video source (WEB, RAW, BD, DVD, etc.)
  </ResponseField>
  
  <ResponseField name="nc" type="boolean">
    Creditless flag
  </ResponseField>
  
  <ResponseField name="subbed" type="boolean">
    Hardsubbed flag
  </ResponseField>
  
  <ResponseField name="lyrics" type="boolean">
    On-screen lyrics flag
  </ResponseField>
  
  <ResponseField name="uncen" type="boolean">
    Uncensored flag
  </ResponseField>
  
  <ResponseField name="overlap" type="string">
    Overlap type (None, Transition, Over)
  </ResponseField>
  
  <ResponseField name="audio_id" type="integer" optional>

  </ResponseField>
  
  <ResponseField name="link" type="string">
    Direct streaming URL

  <ResponseField name="tags" type="string">
    Concatenated tags (e.g., "NCBD1080")

  <ResponseField name="created_at" type="string">

  </ResponseField>
  
  <ResponseField name="updated_at" type="string">
    ISO 8601 timestamp
  </ResponseField>

## Show Video

<ParamField path="GET" type="endpoint">
  /api/video/:basename
</ParamField>

Retrieve a single video by its basename.

### Path Parameters

<ParamField path="basename" type="string" required>
  The video basename (filename without extension)
</ParamField>

### Query Parameters

<ParamField query="include" type="string">
  Same includes as list endpoint
</ParamField>

### Example Request

```bash
curl "https://api.animethemes.moe/api/video/CowboyBebop-OP1?include=animethemeentries.animetheme.anime,audio,videoscript"
```
